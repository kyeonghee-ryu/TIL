# GIT-INTERMEDIATE



## Create a Branch

### 브랜치생성하기

```
$ git branch about # about이라는 이름의 branch 생성 
```



### 브랜치 확인하기(git branch)

```
$ git branch
```



### 브랜치 옮기기 (head 옮기기)

```
$ git checkout about
```



### 브랜치 합치기 (git merge)

 -항상 master에서 다른 브랜치를 병합

```
$ git merge about # about을 master와 병합
```



### 브랜치 삭제하기

```
$ git branch -d about
```
