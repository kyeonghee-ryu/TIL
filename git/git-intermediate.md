# GIT-INTERMEDIATE

## Create a Repository

1. https://github.com에서 Repositories New를 클릭한다.
2. Repository 이름을 설정하고 , README를 체크한다

## Create a Branch

### 브랜치생성하기 (git branch <name>)

### 브랜치 확인하기(git branch)

### 브랜치 옮기기 (head 옮기기 git switch)

### 브랜치 합치기 (git merge)

 -항상 master에서 다른 브랜치를 병합

### 브랜치 삭제하기(git  branch -d <name)
